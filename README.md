# 迷宫大挑战

迷宫大挑战是一个休闲类游戏，玩家操控车标在2D俯视角迷宫中移动，通过选择前进方向、利用道路层叠、并遵守相关规则，完成从起点到终点的既定路径。

# 下载&运行&更新

---

1. 下载MazeChallenge.zip并解压
2. 运行launch.bat

删除旧版所有文件并运行新版文件即可保留数据更新

# 注意事项

---

1. 关闭游戏时，使用右上角关闭按钮，请勿使用其他软件强制关闭
2. 用户数据保存在本地（当前用户家目录下），请勿手动修改
